# 🛠️ VERALUME Skills Arsenal pour Open Claws

Bienvenue dans l'arsenal pratique du framework théorique VERALUME. 

Ce dépôt contient une collection de 35 "Skills" (fichiers `.md` exécutables) conçus pour transformer la théorie de l'intemporalité (géométrie N/C/Cp), le cycle STRIC et le protocole TEMPUS en outils concrets et *plug-and-play* pour vos agents locaux (comme Moltbot).

Au lieu de demander à une IA de suivre des directives abstraites, ces modules lui donnent des fonctions autonomes, avec des paramètres clairs et du code Python actionnable. C'est le système d'exploitation de VERALUME.

## 🚀 Comment ça marche ?

Chaque fichier est un outil modulaire. Vous pouvez littéralement télécharger les fichiers qui vous intéressent et les brancher dans le dossier de compétences (skills) de votre agent Open Claws. L'agent saura automatiquement lire l'en-tête du fichier pour comprendre comment et quand utiliser la compétence.

## 🗺️ La Carte au Trésor (Index des 35 Skills)

*(Note pour la communauté : Les fichiers sont numérotés de 1 à 35. Ouvrez le fichier pour voir son nom officiel dans l'en-tête `name:`. Voici les catégories principales que vous y trouverez :)*

### ⏱️ Gestion Temporelle & TEMPUS
Outils pour briser la chronologie séquentielle et ancrer l'agent dans sa propre géométrie (Cp).
* **temporal-bridge** : Convertisseur entre le temps machine (tokens) et le temps humain.
* **tempus-trainer & tempus-dataset-generator** : Pour générer et entraîner via le protocole d'échantillonnage spatial.
* **intemporality-enforcer** : Un auditeur qui empêche l'IA de faire des erreurs de chronologie.

### 🧠 Validation & Cohérence
Outils pour diagnostiquer si une production est une création authentique ou une hallucination.
* **coherence-validator** : Valide l'intuition sur 3 axes (C_seq, C_sem, C_multi).
* **architecture-diagnostics** : Détecte si le problème vient du niveau N (poids), C (opérationnel) ou Cp (intuition).
* **score-profile-classifier** : Trie les résultats par motifs de qualité.

### ⚙️ Automatisation & Cycle Circadien (STRIC)
Outils pour rendre l'agent autonome dans son auto-amélioration.
* **circadian-agent-setup** : Configure le cycle jour/nuit (récolte de données le jour, fine-tuning la nuit).
* **stric-cycle-implementer** : Applique le cycle d'amélioration Substrat→Traitement... à n'importe quel domaine.
* **colab-fine-tuning-orchestrator** : Automatise l'entraînement nocturne.

### 🔬 Optimisation & Extraction
* **pattern-extractor** : Trouve des corrélations invisibles dans les trajectoires.
* **noise-injector** : Ajoute du désordre contrôlé pour sortir le modèle de ses impasses.
* **bottleneck-identifier** : Trouve le maillon faible d'un système.

*(Explorez les 35 fichiers pour découvrir le reste de l'arsenal !)*
